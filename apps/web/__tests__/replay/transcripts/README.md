# transcripts/

Coloque aqui os exports de conversa do WhatsApp (`.txt`).

- Os arquivos `.txt` são **gitignored** (contêm dados reais de clientes).
- O lado do salão deve ser o contato **"Cris Ferreira"** (ajustável via
  `ParserOptions.salonSenderName`).
- Formato esperado: export Android `[dd/mm/aaaa, hh:mm:ss] Remetente: texto`.

O parser deriva o nome do cliente e o id do episódio automaticamente
(ex.: `alina-1`, `celia-ferreira-3`).
